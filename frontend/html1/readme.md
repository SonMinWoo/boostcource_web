# HTML TAG 및 배치 

* html태그는 기본적으로 head, body로 나누어진다.
* head에는 타이틀(페이지의 이름)과 웹페이지의 기본적인 정보, css 스타일 등이 입력된다.
* body는 웹페이지의 내용이 들어간다.

## 코드의 태그 설명

* meta : 웹페이지의 기본적인 정보
* title : 웹페이지의 이름

**body는 header와 중간 내용, footer로 구성된다.**

* header : body에서 상단부분을 차지한다.
* nav : 링크를 보여주는 페이지의 한 구획을 말한다. 주로 태그 안에 메뉴, 목차, 색인 등이 들어간다.
* ul : 순서 없는 list를 나타낸다. li 태그가 자식으로 온다. _(cf>ol은 순서있는 list)_

* section : 일반적인 구획을 나타낸다. 무분별한 div 사용을 지양하고 본 태그를 사용하자.
* button : 말 그대로 버튼을 만드는 태그이다. 내부에 택스트나 이미지 등의 내용을 넣을 수 있다.
* div : id나 class를 줘서 레이아웃 설계에 쓰인다. 
* img src, alt : 이미지의 소스 링크, 설명이다.
 
* footer에는 웹페이지에서 아래쪽에 위치하는 내용이 들어간다.
* span : div와 동일한 기능을 가진다. 단, div는 block속성이고, span은 inline속성이다.
